SCSS - CSS con Superpoderes

Sabemos que antes se podian hacer cosas increibles con CSS, pero ahora a llegado un CSS con Superpoderes llamado SCSS, este viene siendo lo mismo que CSS pero mejorado en muchas funciones. Ahora podemos utilizar variables, guardar datos dentro de otros usando la anidación, tenemos más variedad en propiedades, Etc... Este lenguaje se creo con el fin de mejorar el modo con el que se escribia CSS a SCSS, espero que los que esten practicando este lenguaje lo perfeccionen y lo utilisen al 100%.

Aún hay alguien que le hace competencia SCSS

Sí mi querido amigo, y ese contrincante es SASS, te preguntaras ¿Qúe es SASS?, bueno SASS es la 2da forma de escribir CSS con si fuera SCSS, la diferencia es que SCSS se escribe de una manera mientras SASS se escribe de otra, ahora ¿Cómo se escriben?, sabemos que en CSS teniamos que poner ese molesto punto y coma y las llaves ( ; { } ) en cada propiedad, bueno de esa manera se escribe SCSS, agregando ese punto y coma al final, pero SASS ya cambia la historia, con SASS nos desasemos de ese molesto puento y coma, lo único que agregamos son los dos puntos en una propiedad (margin-top: 20px), bueno pues asi se se escribe SASS. Estas son dos Syntax de verdad bastante usadas y tu puedes escojer la que más te guste, yo personalmente uso SCSS porque me parece que con los puntos y comas y llaves puedo leer mejor un documento.
